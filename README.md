# CarouBot
An Automated Carousell Listing Tracker

Automatically scrapes Carousell's search API every few minutes to check for new listings with keywords, and notifies in Discord Channel for new listings. 

## Features
- Ability to set target search terms for scanning
- Prevents duplicate notifications by storing listing ID
